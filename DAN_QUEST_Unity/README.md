# DAN QUEST Unity Migration

This Unity project is the foundation migration for the existing DAN QUEST idea.

Open `DAN_QUEST_Unity` in Unity. On first editor load, `Assets/Scripts/Editor/DanQuestUnityFoundationGenerator.cs` generates:

- character ScriptableObjects
- clean 3D proxy prefabs
- portrait sprite import settings
- `DAN_QUEST_CharacterSelect`
- `DAN_QUEST_Arena_Test`
- build settings for both scenes

The uploaded character images are used only as portraits/reference UI, not as gameplay cutouts.
